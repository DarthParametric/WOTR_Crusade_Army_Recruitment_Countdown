# About
A mod for Owlcat's Pathfinder: Wrath of the Righteous. Adds a message to the combat log with the remaining days until crusade army recruitment is available.

# Installation:
1. Download and install [Unity Mod Manager](https://www.nexusmods.com/site/mods/21) and set it up for WOTR ("Pathfinder Second Adventure").
1. Download the latest version of [Crusade Army Recruitment Countdown](https://github.com/DarthParametric/WOTR_Crusade_Army_Recruitment_Countdown/releases/latest).
1. Drag the mod zip into Unity Mod Manager or install manually.
1. Run your game.
